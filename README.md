# NoteIt

Full-stack note-taking app with role-based auth, CRUD notes, sharing, and publishing.

## Dev Setup

Two terminals:

**Terminal 1 — Backend:**
```bash
cd backend
npm install
npm run dev
```
Runs on http://localhost:5000

**Terminal 2 — Frontend:**
```bash
cd frontend
npm install
npm run dev
```
Runs on http://localhost:5173

## Requirements
- Node.js 18+
- MongoDB running locally on port 27017
